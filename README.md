# casino2016

## Development Mode

### Run application:

To run from command-line, do
```
lein run
```

### Figwheel:

To run from repl with figwheel (after cider-jack-in), do:
```
user=> (start)
```
This function is defined in the ```dev/user.clj``` file.

Another option is via lein:
```
lein figwheel
```
